## 기능 요구사항
1. 좌표값 입력 : 에러 출력 후 다시 입력
    - 괄호 분리 -> 괄호가 아닌 경우 에러
    - 쉼표 분리 -> 쉼표가 아닌 경우 에러
    - 구분자 '-' 분리 -> 구분자 아닌 경우 에러 
    - 숫자 0~24 인지 검증 -> 아닌 경우 에러
2. 거리 계산 출력

## 프로그래밍 요구사항
1. depth 1로 구현
2. else 사용 금지
3. 메서드 길이 10줄 넘어가지 않도록 구현
4. 메소드 한가지 일만 하도록 구현
5. 일급 컬렉션 사용

**\* 상속 또는 인터페이스 사용해서 구현**

## 실행 결과
```
좌표를 입력하세요.
(10,10)-(14,15)
두 점 사이 거리는 6.403124
```
